FloatingMD 图片功能测试

📝 编辑器访问

请选择要测试的编辑器：

独立编辑器 浮动编辑器 (需要在网页中测试)

🖼️ 新增图片功能

支持的功能：

  • 📋 图片粘贴：复制图片后按 Ctrl+V 粘贴
  • 🖱️ 图片拖拽：直接拖拽图片文件到编辑器
  • 📁 文件选择：点击"图片"按钮选择本地文件
  • 🔍 图片预览：点击图片可放大查看
  • 💾 自动存储：图片自动保存到浏览器存储
  • 📱 自适应显示：图片尺寸自动适配编辑器宽度

🧪 测试步骤

  1. 打开编辑器：点击上方的编辑器链接
  2. 测试粘贴功能：
    • 在其他应用中复制一张图片（如截图工具、浏览器图片等）
    • 在编辑器中按 Ctrl+V
    • 图片应该自动插入到编辑器中
  3. 测试拖拽功能：
    • 从文件管理器中拖拽一张图片文件
    • 拖拽到编辑器区域并释放
    • 图片应该自动插入
  4. 测试文件选择：
    • 点击工具栏中的"图片"按钮
    • 选择本地图片文件
    • 图片应该插入到编辑器中
  5. 测试预览功能：
    • 点击编辑器中的图片
    • 应该弹出放大预览窗口
    • 点击预览窗口外部可关闭
  6. 测试存储功能：
    • 插入图片后刷新页面
    • 图片应该仍然存在

💡 支持的图片格式

  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • GIF (.gif)
  • WebP (.webp)
  • BMP (.bmp)
📌 注意事项：
  • 图片会自动压缩到最大宽度800px以优化性能
  • 图片数据存储在浏览器本地存储中
  • 浮动编辑器需要在实际网页中测试（Chrome扩展环境）
  • 确保浏览器允许访问剪贴板和文件
